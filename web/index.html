                    Product thesis
                    
A writing workspace for the work after capture.

                    

                        EverFree is for the moment when you have notes, scraps, research, and an unfinished draft. The assistant lives beside the editor so the next paragraph is easier to write.
                    

                

                

                    

                        editor
                        Three-pane notes, search, a real writing surface, voice input, image uploads, and GitHub sync.
                    

                    

                        assistant
                        Ask about the current note, attach selected text, search the web, read pages, or generate an image with `/image`.
                    

                    

                        import
                        Evernote sync is included in the Mac app, so the old archive becomes writing context.
                    

                    

                        own
                        Free MIT-licensed software, source on GitHub, and a signed/notarized DMG you can download and run.
                    

                

            


            

                
Download the Mac app when you want Evernote import and the full local assistant. Use the web editor when you want the same writing workspace from the browser.
